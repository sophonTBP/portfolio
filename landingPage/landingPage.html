This project is part of the free-code-camp curriculum and is my attempt to go a little bit beyond the exercice requirements. The code is available on github (link) as well a links to various video tutorials I used to build it.

The NavBar is responsive but doesn't use a hamburger menu, instead icons replace the text labels on lower screen resolutions. Every button as some kind of animation to indicate interactivity.

The theme button lets the user chose between preselected colors(fireship link). Svgs allow a lot of customisation as they are viewed by the browser just like any other "div". One can easily* change the color of any sub element. Couple that with css variables, a few DOM selectors and a little bit of javascript magic and "voila!". You can find free Svgs on google, I found the ones I used on "Undraw". *you may need a tool like "inkscape" to edit the svg and rename the path or the group you want to change the color of.

Scalable Vector Graphics is an Extensible Markup Language-based vector image format for two-dimensional graphics with support for interactivity and animation. The SVG specification is an open standard developed by the World Wide Web Consortium since 1999. SVG images and their behaviors are defined in XML text files
